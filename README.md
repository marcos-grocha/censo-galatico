# Censo Galático 🌌

O universo de **Star Wars** é composto de uma galáxia com inúmeros planetas e espécies, nesta aplicação podemos buscar esses dados através de consultas em **APIs externas**.

---

## Funcionalidades

- **Visualização de Planetas**: Informações sobre nome, clima, população e terreno.  
- **Visualização de Habitantes Famosos**: Informações sobre nome e ano de nascimento.  

---

## Tecnologias Utilizadas

- **JavaScript**
- **HTML**
- **CSS**
